# Statistics Calculator Implementations

This project contains three implementations of a basic statistics calculator (Mean, Median, Mode) demonstrating different programming paradigms:

* **C (`statistics.c`):** Procedural approach.
* **OCaml (`statistics.ml`):** Functional approach.
* **Python (`statistics.py`):** Object-Oriented approach.

## Prerequisites
Ensure you have the following installed on your system:
- **C:** A C compiler (e.g., `gcc`).
- **OCaml:** [OPAM](https://opam.ocaml.org/) and `dune` (`opam install dune`).
- **Python:** Python 3.x.

## How to Run

### 1. C (Procedural)
Compile and run the program using `gcc`:
```bash
gcc statistics.c -o statistics
./statistics
```
### 2. C (Funcitonal)
```bash
dune exec ./statistics.exe
```

```bash
### . Python (OOP)
python statistics.py
```
