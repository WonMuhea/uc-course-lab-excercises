
#  Generating and activating the root venv
## MacOS/Linux
cd ai-assignments
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip

## Windows
cd ai-assignments
py -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip

1. First run this read me to make the virtual env ready for the projects
